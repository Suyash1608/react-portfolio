# 🚀 Suyash Gupta — Developer Portfolio

A modern, responsive personal portfolio built with **React + Vite**, featuring a dark glassmorphism design. Showcases my skills, projects, experience, education, and certifications as a Java Backend Developer & Corporate Trainer.

---

## ✨ Features

- ⚡ Built with **React 19** + **Vite** for blazing-fast performance
- 🎨 Dark theme with **glassmorphism** UI & smooth animations
- 📱 Fully **responsive** across all devices
- 🔗 Dynamic routing via **React Router v7**
- 📂 Data-driven architecture — all content managed via `src/assets/` files
- 🧭 Sticky navbar with active route highlighting
- 🗂️ Sections: Home, About, Projects, Skills, Contact

## 📁 Project Structure


## 🚀 Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Suyash1608/react-portfolio.git

# Navigate into the project directory
cd react-portfolio-main

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be running at **http://localhost:5173**

---

## 🔧 Customisation

All portfolio content is data-driven. To update your own information, edit the files inside `src/assets/`:

| File                  | Content                              |
|-----------------------|--------------------------------------|
| `personalData.js`     | Name, role, bio, contact info        |
| `projects.js`         | Java projects & academic projects    |
| `skillsData.js`       | Skills grouped by category           |
| `experienceData.js`   | Work experience entries              |
| `educationData.js`    | Education history                    |
| `certificationData.js`| Certifications                       |
| `trainingData.js`     | Training / courses                   |
